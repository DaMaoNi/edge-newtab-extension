   或直接下载 ZIP 文件并解压

2. **打开扩展管理页面**
   - **Edge**: 访问 `edge://extensions/`
   - **Chrome**: 访问 `chrome://extensions/`

3. **开启开发者模式**
   - 在页面右上角打开"开发者模式"开关

4. **加载扩展**
   - 点击"加载解压缩的扩展"按钮
   - 选择项目文件夹（包含 manifest.json 的目录）

5. **完成安装**
   - 打开新标签页即可看到效果！
