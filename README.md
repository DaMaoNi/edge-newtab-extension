## ✨ 功能特性

- 🖼️ **随机壁纸** - 每次打开新标签页自动加载高清随机壁纸
- 🕐 **实时时钟** - 显示当前时间和日期，简洁美观
- 🔍 **快速搜索** - 使用浏览器默认搜索引擎快速搜索
- 🚀 **快捷导航** - 常用网站快捷入口，一键直达
- 🔄 **一键换图** - 点击按钮或按 R 键快速更换壁纸
- ⚡ **轻量高效** - 纯原生实现，无依赖，加载迅速

## 🚀 安装方法

### 方法一：开发者模式加载（推荐）

1. **下载项目**
      或直接下载 ZIP 文件并解压

2. **打开扩展管理页面**
   - **Edge**: 访问 `edge://extensions/`
   - **Chrome**: 访问 `chrome://extensions/`

3. **开启开发者模式**
   - 在页面右上角打开"开发者模式"开关

4. **加载扩展**
   - 点击"加载解压缩的扩展"按钮
   - 选择项目文件夹（包含 manifest.json 的目录）

5. **完成安装**
   - 打开新标签页即可看到效果！

## 📖 使用说明

### 基本功能

| 功能 | 说明 |
|------|------|
| 查看时间 | 页面顶部显示实时时钟和日期 |
| 搜索 | 在搜索框输入内容，按回车或点击搜索按钮 |
| 快捷访问 | 点击下方的快捷图标访问常用网站 |
| 更换壁纸 | 点击右下角的刷新按钮或按键盘 `R` 键 |

### 快捷键

- `R` - 快速更换壁纸（非输入状态时）
- `Enter` - 执行搜索

- 雪球
- 哔哩哔哩
- GitHub
